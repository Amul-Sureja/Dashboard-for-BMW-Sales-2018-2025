## 📊 Project Overview

This dashboard provides an **interactive analysis of BMW global sales data (2018–2025)**.

### ✨ Features

- 📍 Region-wise and model-wise analysis
- 📈 Monthly and yearly sales trends
- ⚡ EV adoption insights (BEV Share)
- 📦 Distribution analysis (Histogram, Box Plot)
- 🔗 Correlation Heatmap
- 🎯 Dynamic filtering system

---

## 🛠 Tech Stack

- **Frontend:** Dash (Python)
- **Visualization:** Plotly
- **Data Processing:** Pandas
- **Styling:** CSS
